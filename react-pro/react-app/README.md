# React + TypeScript + Vite


## Commandas

- Install:

```bash
npm create vite@latest
```

- Add the **name** of the project, the **framework** (React) and a **Variant** (TypeScript + SWC).

- Go to the app folder, install de dependencies and run it locally with:

```bash
cd react-app
npm install
npm run dev
```
